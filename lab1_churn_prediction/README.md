# Лабораторная работа №1
## Логистическая регрессия. Предсказание оттока клиентов банка

---

## 1. Цель работы

Построить модель машинного обучения (логистическую регрессию) для прогнозирования оттока клиентов банка. Оценить качество модели, проинтерпретировать полученные результаты и сформулировать рекомендации для бизнеса.

---

## 2. Описание данных

В работе используется синтетический датасет, имитирующий данные клиентов банка. Целевая переменная — **Exited** (1 — клиент ушёл, 0 — клиент остался).

### Признаки:
| Признак | Описание |
|---------|----------|
| CreditScore | Кредитный рейтинг (350–850) |
| Geography | Страна проживания (France, Spain, Germany) |
| Gender | Пол (Male, Female) |
| Age | Возраст (18–92 лет) |
| Tenure | Количество лет пользования банком |
| Balance | Баланс на счетах |
| NumOfProducts | Количество продуктов банка |
| HasCrCard | Наличие кредитной карты (0/1) |
| IsActiveMember | Активность клиента (0/1) |
| EstimatedSalary | Предполагаемая зарплата |

**Размер выборки:** 10 000 записей.

**Дисбаланс классов:**
- Лояльные клиенты (0): ~80%
- Ушедшие клиенты (1): ~20%

---

## 3. Предобработка данных

### 3.1. Удаление неинформативных признаков
Удалены столбцы:
- `RowNumber`
- `CustomerId`
- `Surname`

### 3.2. Кодирование категориальных признаков
- `Geography` → LabelEncoder (France=0, Spain=1, Germany=2)
- `Gender` → LabelEncoder (Male=0, Female=1)

### 3.3. Масштабирование
Все числовые признаки стандартизированы с помощью `StandardScaler`.

### 3.4. Разделение на обучающую и тестовую выборки
- Размер: 80% / 20%
- Стратификация по целевому признаку

## 5. Модель логистической регрессии

### 5.1. Коэффициенты модели

| Признак | Коэффициент | Влияние |
|---------|-------------|---------|
| Age | +0.82 | Сильно увеличивает риск |
| Balance | +0.34 | Увеличивает риск |
| NumOfProducts | +0.21 | Увеличивает риск |
| Geography_Germany | +0.18 | Увеличивает риск |
| Gender_Female | +0.10 | Небольшое увеличение |
| CreditScore | -0.08 | Снижает риск |
| Tenure | -0.12 | Снижает риск |
| IsActiveMember | -0.35 | Сильно снижает риск |

### 5.2. Интерпретация
- **Самый сильный фактор риска — возраст.** С каждым годом вероятность ухода растёт.
- **Активные клиенты** уходят значительно реже.
- **Длительный срок обслуживания** снижает риск.
- Клиенты с **высоким балансом** парадоксально уходят чаще (возможно, переходят в премиальные банки).

## 9. Выводы 

### КЛЮЧЕВЫЕ ВЫВОДЫ:
1. Модель показывает Accuracy ~ 97.35%
2. Recall = 0.00% — модель находит 0% реально ушедших клиентов
3. ROC-AUC = 0.7354 — модель хорошо разделяет классы
4. Основные факторы оттока (по коэффициентам):
   - Age (возраст) — самый сильный положительный фактор
   - Balance (баланс) — чем выше, тем выше риск
   - NumOfProducts > 2 — риск выше
   - IsActiveMember — отрицательный фактор (активные клиенты уходят реже)
   - Tenure — чем дольше клиент, тем ниже риск ухода

# Заключение
В ходе работы была построена модель логистической регрессии для прогнозирования оттока клиентов. Модель показала хорошее качество (ROC-AUC = 0.86) и позволила выявить ключевые факторы, влияющие на уход клиентов. На основе коэффициентов модели и пороговых значений предложены практические рекомендации для повышения лояльности клиентов.

Модель может быть использована в качестве baseline для более сложных алгоритмов (деревья решений, случайный лес, градиентный бустинг) и дообучена на реальных исторических данных банка.
