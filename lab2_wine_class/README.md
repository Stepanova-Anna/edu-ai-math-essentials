### **1. Загрузка и подготовка данных**

Для выполнения работы был использован датасет `load_wine()` из библиотеки `sklearn.datasets`. Датасет содержит 178 образцов вина, каждый из которых описан 13 признаками. Целевая переменная представляет собой 3 класса вин.

**Этапы подготовки данных:**
1.  Целевая переменная (`y_data`) была преобразована в формат one-hot encoding с помощью `utils.to_categorical`.
2.  Данные были разделены на три выборки в следующем соотношении:
    *   **Обучающая выборка:** 144 образца
    *   **Валидационная выборка:** 16 образцов
    *   **Тестовая выборка:** 18 образцов
3.  Для ускорения сходимости модели и улучшения ее работы все признаки были стандартизированы с помощью `StandardScaler`. Масштабирование проводилось на основе обучающей выборки, а затем применено к валидационной и тестовой выборкам.

---

### **2. Архитектура нейронной сети**

Для решения задачи была создана полносвязная нейронная сеть (Sequential) со следующей архитектурой:

*   **Входной слой:** 13 нейронов (по числу признаков).
*   **Скрытый слой 1:** 128 нейронов, функция активации ReLU. После слоя применяются Batch Normalization и Dropout с коэффициентом 0.3 для регуляризации.
*   **Скрытый слой 2:** 64 нейрона, функция активации ReLU. После слоя применяются Batch Normalization и Dropout с коэффициентом 0.2.
*   **Скрытый слой 3:** 32 нейрона, функция активации ReLU. После слоя применяются Batch Normalization и Dropout с коэффициентом 0.1.
*   **Выходной слой:** 3 нейрона, функция активации Softmax для многоклассовой классификации.

**Общее количество параметров модели:** 13,123.

### **3. Компиляция и обучение модели**

*   **Оптимизатор:** Adam с learning rate = 0.001.
*   **Функция потерь:** `categorical_crossentropy`.
*   **Метрика:** `accuracy`.
*   **Количество эпох:** 500 (с использованием ранней остановки).
*   **Размер батча (batch size):** 16.
*   **Callbacks:**
    *   `EarlyStopping` с `patience=50` и `restore_best_weights=True` для предотвращения переобучения и сохранения лучшей модели на валидации.
