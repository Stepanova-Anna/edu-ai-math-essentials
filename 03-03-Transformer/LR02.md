# Отчет по лабораторной работе №2 (блок Transformer)
## Transformer encoder для классификации отзывов IMDB

---

## Цель работы

Перенести архитектуру Transformer encoder, разработанную в ЛР01 на синтетических данных, на реальную задачу бинарной классификации отзывов IMDB (положительный/отрицательный сентимент).

---

## Основные изменения по сравнению с ЛР01

| Аспект | ЛР01 (synthetic) | ЛР02 (IMDB) |
|--------|------------------|-------------|
| Данные | Случайные последовательности | Реальные отзывы |
| Токены | 1-9, 7 и 3 для порядка | Слова → индексы из словаря |
| Длина | SEQ_LEN = 12 | MAXLEN = 200 |
| Размер выборки | 5000 примеров | 5000 train / 2000 test |
| Требуемая точность | ≥ 0.95 | ≥ 0.75 (на subset) |

# Вывод

В ходе лабораторной работы успешно выполнена трансляция архитектуры Transformer encoder с синтетической задачи на реальные данные IMDB. Ключевые выводы:

Transfer learning архитектуры — код из ЛР01 (TokenAndPositionEmbedding, TransformerEncoderBlock, masked_average) полностью переиспользуется без изменений.

Реальная эффективность — модель достигает точности >75% на подмножестве IMDB всего за 3 эпохи, что подтверждает применимость Transformer для классификации текстов.

Практические аспекты — важность padding маски (чтобы не учитывать [PAD] токены), необходимость обрезания heatmap для читаемости, работа со словарём IMDB и обратным декодированием.

Модель успешно различает положительные и отрицательные отзывы, а визуализация внимания позволяет интерпретировать, на какие части отзыва модель обращает больше внимания.
