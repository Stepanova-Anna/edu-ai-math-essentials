## 3. Реализация MLP с нуля

### 3.1. Архитектура MLP

Реализован класс `MLP` с поддержкой:
- Произвольной архитектуры (1-2 скрытых слоя)
- Функций активации: ReLU, tanh
- Мини-батчевого SGD
- L2-регуляризации
- Сигмоиды на выходе для бинарной классификации
- Отслеживания истории потерь и точности

### 3.2. Основные методы

| Метод | Описание |
|-------|----------|
| `forward(X)` | Прямой проход с кэшированием промежуточных значений |
| `backward(X, y, y_pred)` | Обратный проход с вычислением градиентов |
| `update_weights(gradients_w, gradients_b)` | Обновление весов и смещений |
| `fit(X_train, y_train, X_val, y_val, verbose)` | Обучение модели с мини-батчевым SGD |
| `predict(X, threshold)` | Предсказание классов |
| `predict_proba(X)` | Предсказание вероятностей |

### 3.3. Математическая реализация

**Прямой проход:**

z^(l) = a^(l-1) · W^(l) + b^(l)
a^(l) = activation(z^(l))

Для скрытых слоёв используется ReLU: `a = max(0, z)`

**Выходной слой (сигмоида):**

z_out = a^(L-1) · W^(L) + b^(L)
ŷ = σ(z_out) = 1 / (1 + e^(-z_out))


**Функция потерь (бинарная кросс-энтропия):**

L = -1/m · Σ [y·log(ŷ) + (1-y)·log(1-ŷ)]


**Обратный проход (градиенты):**

δ^(L) = ŷ - y (для выходного слоя)
δ^(l) = (δ^(l+1) · (W^(l+1))^T) ⊙ activation'(z^(l))


**Обновление весов с L2-регуляризацией:**

W^(l) ← W^(l) - η · (∇W^(l) + λ·W^(l)/m)
b^(l) ← b^(l) - η · ∇b^(l)

### 9.1. Интерпретация весов

| Признак | Вес (приблизительно) | Влияние |

### 9.2. Содержательная интерпретация

- **Variance (дисперсия):** Чем выше дисперсия, тем меньше вероятность, что банкнота фальшивая.
- **Skewness (асимметрия):** Отрицательный вес — более симметричные изображения чаще подлинные.
- **Curtosis (эксцесс):** Положительный вес — более остроконечные распределения чаще фальшивые.
- **Entropy (энтропия):** Положительный вес — более хаотичные изображения чаще фальшивые.
