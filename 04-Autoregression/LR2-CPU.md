# Отчет по лабораторной работе №2 (CPU-вариант)
## Декодерный трансформер на Tiny Shakespeare

---

## Цель работы

Перенести авторегрессионную модель предсказания следующего токена с синтетических данных на реальный текстовый корпус Tiny Shakespeare. Реализовать CPU-дружественную версию декодерного трансформера для генерации текста посимвольно.

---

# Вывод

В ходе лабораторной работы реализован декодерный трансформер для символьной генерации текста на корпусе Tiny Shakespeare. Ключевые результаты:

1. **Переносимость архитектуры** — модель, разработанная для синтетических данных (ЛР01), успешно адаптирована к реальному тексту без изменения базовой логики.
2. **Качество** — модель достигла перплексии ~3.16, что значительно лучше частотного ориентира (~4.50), подтверждая способность выявлять языковые паттерны.
3. **Генерация** — в 3+ из 20 фиксированных промптов модель генерирует продолжения с совпадением ≥20% символов с эталоном, средний уровень совпадения ≥10%.
4. **Причинная маска** — диагностика подтвердила отсутствие доступа к будущим позициям (future_ratio < 1e-4).
