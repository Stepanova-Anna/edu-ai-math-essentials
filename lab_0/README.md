---

#### Задание 1.2: Операции с векторами

**Сложение векторов:**
```python
sum_ab = a + b                # [4, 6]
```
Сложение векторов выполняется поэлементно. Результат геометрически соответствует диагонали параллелограмма, построенного на векторах `a` и `b`.

**Умножение на скаляр:**
```python
scaled_a = 2 * a              # [6, 8]
```
Умножение вектора на скаляр изменяет его длину без изменения направления (при положительном скаляре).

---

#### Задание 1.3: Скалярное произведение и норма

**Скалярное произведение:**
```python
dot_product = np.dot(a, b)    # 11
```
Скалярное произведение показывает степень сонаправленности векторов.

**Норма (длина) вектора:**
```python
norm_a = np.linalg.norm(a)    # 5.000
norm_b = np.linalg.norm(b)    # 2.236
```

**Угол между векторами:**
```python
cos_angle = dot_product / (norm_a * norm_b)  # 0.984
angle_deg = np.degrees(np.arccos(cos_angle)) # 10.3°
```
Угол между векторами `a` и `b` составляет **10.3°**, что подтверждает их высокую степень сонаправленности.

**Созданные матрицы:**

A = np.array([[1, 2], [3, 4]])          # 2x2

---

#### Задание 2.2: Операции с матрицами

**Сложение матриц:**
```python
sum_AB = A + B
```
Результат:
```
[[ 6  8]
 [10 12]]
```

**Умножение матрицы на скаляр:**
```python
scaled_A = 3 * A
```
Результат:
```
[[ 3  6]
 [ 9 12]]
```

**Умножение матриц:**
```python
product_AB = A @ B
```
Результат:
```
[[19 22]
 [43 50]]
```

**Проверка некоммутативности:**
```python
product_BA = B @ A
```
Результат:
```
[[23 34]
 [31 46]]
```
`A*B != B*A` — умножение матриц **не коммутативно**.

---

#### Задание 2.3: Умножение матрицы на вектор

```python
result_Aa = A @ a  # [11, 25]
```
Геометрический смысл: матрица `A` действует как линейное преобразование, переводящее вектор `a` в новый вектор `[11, 25]`. Это демонстрирует, как матрицы могут выполнять повороты, масштабирование и другие преобразования пространства.

#### Задание 2.4: Транспонирование и специальные матрицы

**Транспонирование:**
```python
A_T = A.T
```
Результат:
```
[[1 3]
 [2 4]]
```

**Специальные матрицы:**
- **Единичная матрица** (`np.eye(3)`): диагональная матрица с 1 на главной диагонали
- **Нулевая матрица** (`np.zeros((2, 3))`): все элементы равны 0
- **Матрица из единиц** (`np.ones((2, 2))`): все элементы равны 1

#### Задание 3.3: Простая линейная регрессия с матрицами

Была реализована модель линейной регрессии для предсказания цены дома по площади, используя метод наименьших квадратов.

**Данные:**
- Площади: `[50, 60, 70, 80, 90, 100, 110, 120]` (кв.м)
- Цены: `[3.2, 3.8, 4.1, 4.7, 5.2, 5.8, 6.1, 6.9]` (млн руб.)

**Решение:**
```python
w = (X^T * X)^(-1) * X^T * y
```

**Результаты:**
- Свободный член (intercept): `0.624`
- Коэффициент при площади: `0.051`
- Уравнение модели: `Цена = 0.624 + 0.051 × Площадь`
- Предсказание для дома 95 кв.м: `5.49 млн руб.`

#### Задание 1: Евклидово расстояние

```python
def euclidean_distance(vec1, vec2):
    return np.sqrt(np.sum((vec1 - vec2) ** 2))
```
Евклидово расстояние — это длина прямой между двумя точками в пространстве. Широко используется в задачах кластеризации и классификации.

#### Задание 2: Нормализация вектора

```python
def normalize_vector(vec):
    norm = np.linalg.norm(vec)
    return vec / norm if norm != 0 else vec
```
Нормализация приводит вектор к единичной длине, сохраняя его направление. Используется в задачах, где важна только ориентация вектора (например, косинусное сходство).

#### Задание 5: Исследование различных расстояний

Были исследованы следующие метрики расстояний:

| Метрика | Формула | Значение |
|---------|---------|----------|
| Евклидово (L2) | √Σ(xᵢ - yᵢ)² | 2.0680 |
| Манхэттенское (L1) | Σ|xᵢ - yᵢ| | 3.9206 |
| Чебышёва (L∞) | max|xᵢ - yᵢ| | 1.3246 |
| Косинусное | 1 - cos(θ) | 0.8183 |
| Корреляционное | 1 - ρ | 0.9746 |
| Минковского (p=3) | (Σ|xᵢ - yᵢ|³)^(1/3) | 1.7243 |

**Применение в Data Science:**
- **Евклидово (L2):** кластеризация, метод k-средних, регрессия
- **Манхэттенское (L1):** устойчиво к выбросам, LASSO-регрессия
- **Чебышёва (L∞):** задачи оптимизации
- **Косинусное:** анализ текстов, поиск похожих документов
- **Корреляционное:** анализ связей между признаками
- **Минковского:** обобщение L1 и L2 с параметром p
