# Отчет по лабораторной работе №1 (блок Attention)
## GRU seq2seq с Luong attention для reverse-sequence

---

## Цель работы

Реализовать seq2seq модель с механизмом внимания (Luong attention), которая разворачивает входную последовательность в обратном порядке. Attention позволяет декодеру обращаться ко всем позициям энкодера на каждом шаге, а не только к финальному состоянию.

# Вывод

В ходе лабораторной работы реализована seq2seq модель с Luong attention. Ключевое отличие от plain seq2seq — декодер теперь может обращаться ко всем позициям энкодера на каждом шаге через механизм внимания. Это снимает bottleneck фиксированного контекста и улучшает качество на длинных последовательностях. На практике модель успешно научилась обратному порядку токенов, а heatmap внимания показала ожидаемый паттерн смещения фокуса по антидиагонали.
